# Johnson Counter

## Description
A 4-bit Johnson Counter implemented in Verilog HDL.

## Features
- Positive edge-triggered
- Active-high asynchronous reset
- 4-bit output
- 8 unique states
- Synthesizable RTL

## State Sequence
0000 → 0001 → 0011 → 0111 → 1111 → 1110 → 1100 → 1000 → 0000

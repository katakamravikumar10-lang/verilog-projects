# Project 16 – 4-bit Synchronous Binary Counter

## Description
This project implements a 4-bit synchronous binary up counter in Verilog.

## Features
- Positive edge-triggered clock
- Asynchronous active-high reset
- Counts from 0 to 15
- Automatically wraps back to 0

## Files
- sync_counter_4bit.v
- tb_sync_counter_4bit.v
- sync_counter.vcd

## Tools
- Icarus Verilog
- GTKWave
